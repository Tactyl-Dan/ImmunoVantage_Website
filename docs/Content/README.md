## How to Edit

1. **Open any markdown file** in a text editor (VS Code, Notepad, etc.)

2. **Find the section you want to edit** - sections are marked with headers like:
   ```
   ## SECTION NAME
   ```

3. **Edit the text** - update titles, descriptions, bullet points, etc.

4. **Keep the structure** - maintain the format (headers, bold markers, lists)

5. **Save the file** and share it back for website updates

---

## Formatting Guide

### Section Headers
```markdown
## SECTION NAME
```
These mark major sections of the page.

### Subsection Headers
```markdown
### Subsection Name
```
These mark items within sections.

### Bold Text
```markdown
**Bold text here**
```
Used for labels and emphasis.

### Lists
```markdown
- Item 1
- Item 2
- Item 3
```
Used for features, bullet points, etc.

### Links
Content shows where links point to, e.g.:
```markdown
**CTA Button:** Contact Us (links to contact.html)
```

## Tips

- **Don't change the structure** - keep section markers intact
- **Don't add HTML** - just edit the plain text content
- **Keep formatting consistent** - match the existing style
- **Note any new sections** - if you want to add entirely new content, describe what you want
